## この章のゴール

> この章を終えると、Git・VS Code・GitHub アカウントの 3 点セットが揃い、AI コーディングを始める準備が整う。

## 揃えるものは 3 つだけ

AI コーディングを始めるために必要な道具は、たった 3 つである。

| 道具                  | 何をするもの？                                 | 費用 |
| --------------------- | ---------------------------------------------- | ---- |
| **Git**               | ファイルの変更履歴を記録するソフト             | 無料 |
| **VS Code**           | コードを書いたり、Git を操作したりするエディタ | 無料 |
| **GitHub アカウント** | 作ったものをインターネットに公開するサービス   | 無料 |

すべて無料である。順番にインストールしていこう。

## ステップ 1: Git のインストール

### Windows の場合

1. ブラウザで [git-scm.com](https://git-scm.com/) を開く
2. 画面に表示される「Download for Windows」ボタンをクリック
3. ダウンロードされたファイル（`Git-X.XX.X-64-bit.exe`）をダブルクリック
4. インストール画面が表示される。 **基本的にはすべて「Next」で進めてよい**
5. 最後に「Install」をクリックして完了

:::message
インストール途中で「デフォルトエディタ」を選ぶ画面が出る。ここでは **Use Visual Studio Code as Git's default editor** を選んでおくと、後の操作が楽になる。
:::

### Mac の場合

Mac には Git が最初から入っている場合が多い。確認してみよう。

1. 「ターミナル」アプリを開く（Spotlight で「ターミナル」と検索）
2. 以下を入力して Enter キーを押す

```bash
git --version
```

3. `git version 2.XX.X` のように表示されれば、すでにインストール済みである

表示されない場合は、画面の指示に従って「コマンドライン・デベロッパー・ツール」をインストールする。ポップアップが出たら「インストール」を押すだけでよい。

## ステップ 2: VS Code のインストール

1. ブラウザで [code.visualstudio.com](https://code.visualstudio.com/) を開く
2. 「Download」ボタンをクリック（OS を自動判別してくれる）
3. ダウンロードされたファイルを実行
4. インストール画面の指示に従って進める

### 日本語化する

VS Code は初期状態では英語である。日本語にするには以下の手順を踏む。

1. VS Code を起動
2. 左側の四角いアイコン（拡張機能）をクリック
3. 検索バーに「Japanese Language Pack」と入力
4. 表示された拡張機能の「Install」をクリック
5. 右下に「Restart」の通知が出たら、クリックして再起動

再起動すると、メニューが日本語になっている。

## ステップ 3: GitHub アカウントの作成

1. ブラウザで [github.com](https://github.com/) を開く
2. 「Sign up」をクリック
3. メールアドレス、パスワード、ユーザー名を入力
4. 認証（パズルのようなもの）をクリア
5. メールに届く認証コードを入力

これでアカウントが作成される。

:::message
ユーザー名は後から変更できるが、 **公開される名前** になる。仕事で使う予定がある場合は、本名やそれに近いものにしておくと良い。
:::

## ステップ 4: Git の初期設定

Git を使い始める前に、「自分の名前」と「メールアドレス」を登録する必要がある。これは、誰が変更を記録したかを識別するためのものである。

VS Code でこの設定を行う。

1. VS Code を開く
2. 上部メニューの「ターミナル」→「新しいターミナル」をクリック
3. 画面下部にターミナル（文字入力画面）が開く
4. 以下の 2 行を **1 行ずつ** 入力して Enter キーを押す

```bash
git config --global user.name "自分の名前"
```

```bash
git config --global user.email "自分のメールアドレス"
```

:::message
ここで入力する名前とメールアドレスは、GitHub のアカウントと同じものにしておくのが望ましい。
:::

### 設定できたか確認する

以下のコマンドで確認できる。

```bash
git config --global --list
```

`user.name=自分の名前` と `user.email=自分のメールアドレス` が表示されれば成功である。

## ここまでの状態

以下の 3 つが揃っていれば、準備完了である。

- [x] Git がインストールされている
- [x] VS Code が日本語で使える状態になっている
- [x] GitHub アカウントが作成されている
- [x] Git に名前とメールアドレスが登録されている

## 裏で何が起きているか（CLI 解説）

この章で使ったコマンドは 2 つだけ。

| コマンド                                  | 意味                               |
| ----------------------------------------- | ---------------------------------- |
| `git config --global user.name "名前"`    | Git に「自分の名前」を登録する     |
| `git config --global user.email "メール"` | Git に「メールアドレス」を登録する |
| `git config --global init.defaultBranch main` | 新しいリポジトリのデフォルトブランチ名を `main` にする |

`--global` は「このパソコン全体で有効」という意味である。一度設定すれば、以降は不要。

:::message alert
**`init.defaultBranch` の設定は必ず行うこと。** Git のバージョンによっては、`git init` したときのブランチ名が `master` になる。GitHub は `main` が標準なので、ここが食い違うと第 5 章の `git push` でエラーになる。上のコマンドを実行しておけば、常に `main` で始まる。
:::
